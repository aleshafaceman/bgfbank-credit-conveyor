# Кредит под залог недвижимости — каталог опций и пакетов

> Источник: лист «Кредит под залог недвижимости» и связанные справочники (LTV, страхование).  
> Назначение: продукт, скоринг (Loginom), ЛК, панель менеджера.

---

## 1. Логика: не «опции», а пакеты

Клиент выбирает **пакет условий** (1 из 2–3).  
Система применяет **модификаторы** и **надбавки** по правилам.  
Менеджер не редактирует проценты вручную — только подтверждает пакет или эскалирует.

## 4. Модификаторы (опции) — каталог

### 4.1. Выбор клиента (пакеты в ЛК)

| Код | Название для клиента | Суть | Ставка | LTV | Комиссия | Страхование | Кто выбирает |
|-----|----------------------|------|--------|-----|----------|-------------|--------------|
| `PKG_RECOMMENDED` | Рекомендуем | Полный пакет: ККС + стандартная комиссия | База по КИ | По региону | По тарифу | ККС-12 или годовая | Система default |
| `PKG_WITH_INSURANCE` | Со страхованием | Как рекомендуем, акцент на скидку за страхование | −0%…−5%* vs отказ | Без изменений | Стандарт | Обязательно ККС | Клиент |
| `PKG_NO_INSURANCE` | Без страхования жизни/рисков | Отказ от ККС (если политика允许) | **+5%** к базе | Без изменений | Стандарт | Только имущество** | Клиент |
| `PKG_NO_COMMISSION` | Без комиссии за выдачу | Отказ от уплаты комиссии | **+5%** к базе | Без изменений | 0% | По тарифу | Клиент |
| `PKG_SPEC_4_0` | Снижение ставки (лёгкая комиссия) | Спец. Опция 4.0 | Снижение в 1-м периоде | Cap **≤50%** LTV | **0,99%** от суммы | По КИ (пр. 1–2) | Клиент |
| `PKG_SPEC_5_0` | Снижение ставки (стандарт) | Спец. Опция 5.0 | Сильнее снижение | Cap по правилам | **2,49%** (+0,99%*) | По КИ | Клиент |
| `PKG_BUY_RATE` | Купить ставку | Разовая комиссия за минимальную ставку | Мин. по матрице | Без изменений | **5%** | По тарифу | Клиент |
| `PKG_VARIABLE_RATE` | Переменная ставка | Низкая ставка 1–3 мес., далее выше | 1-й период ↓, далее ↑ | Без изменений | По тарифу | По тарифу | Клиент |
| `PKG_FIXED_RATE` | Фиксированная ставка | Отказ от переменной ставки | Стабильная | Без изменений | По тарифу | По тарифу | Клиент |

| `PKG_MANUAL_INCOME` | Подтверждение документами | 2-НДФЛ / справка банка | База + **~0,5 п.п.** | Без изменений | — | — | Клиент (путь входа) |

\*\* «Только имущество» — программа 4 / конструктив, по договору.

## 6. LTV (справочно, не опция)

Определяется: **город + тип ликвидности + тип объекта + тариф + модификаторы**.

| Зона | Пример LTV (квартира) |
|------|------------------------|
| МСК в МКАД, ключевые города МО/ЛО | 55–57% |
| МСК/МО 20–50 км, доп. города | 37–52% |
| Регионы (Казань, Новосибирск, …) | 42–52% |
| Сургут | 35–37% |
| Спец. Опция 4.0 | Не более **50%** |
| Инвест / 6.0 | Не более **45%** |

Повышение LTV в ЛК (`PKG_LTV_BOOST`) — только если скоринг разрешил cap.

## 7. Совместимость (кратко)

| Нельзя вместе | Причина |
|---------------|---------|
| `PKG_SPEC_4_0` + `PKG_SPEC_5_0` | Одна комиссия buy-down |
| `PKG_NO_INSURANCE` + `PKG_WITH_INSURANCE` | Взаимоисключение |
| `PKG_BUY_RATE` + `PKG_NO_COMMISSION` | Политика продукта |
| `PKG_VARIABLE_RATE` + `PKG_FIXED_RATE` | Тип ставки |
| `TURBO_4_0` + сумма < 10 млн | Условие тарифа |
| `PKG_CO_BORROWER` + >2 застрахованных | Лимит ККС |

## 11. Приоритет внедрения в макет
